Start the Studio with Start HOI4 Focus Studio.bat.

Choose Vanilla HOI4 or Road to 56 in the coloured selector at the top. The choice is stored in the project.

For construction rewards that need a province (land fort, coastal fort or naval base), press Open clickable province map. Hover to inspect IDs and click the province you want.

Studio autosaves changes. Backup snapshots are stored under projects/autosaves.

Export is deliberately blocked if a reward was created for the other map profile or the province is not inside the selected state.
