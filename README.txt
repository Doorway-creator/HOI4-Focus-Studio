HOI4 FOCUS STUDIO

1. Double-click "Start HOI4 Focus Studio.bat".
2. The editor opens in your browser. Keep the small launcher window open.
3. Save Project stores your editable work without changing v0.76.
4. Export HOI4 Mod creates a separate folder under "exports".
5. Install Test Build copies a separate test mod directly into your normal
   HOI4 mod folder. Enable it in the launcher and disable older copies.

FOCUS TREE
- Drag a focus to move it.
- Drag empty background space to move around the tree.
- Use the mouse wheel to zoom around the cursor.
- Right-click empty space to place a focus where the cursor is.
- Right-click a focus for connections, icons, and attached events.
- Select a focus to edit its name, ID, description, icon key and raw effects.
- Use Connection type and Connect selected to make prerequisite,
  visual-only, or mutually-exclusive links.
- New Focus creates a blank focus. Delete Selected removes it and its links.
- The Studio grid now uses HOI4's approximate in-game spacing. If two focuses
  overlap in Studio, move them farther apart before exporting.

REWARDS
- Select a focus and use Rewards and Effects in the right sidebar.
- Political values accept positive or negative numbers.
- Duration creates a temporary national modifier.
- National: State Construction lists Norway's states, population, state ID,
  and provinces before adding industry or defensive buildings.
- Technology rewards can immediately unlock a technology ID or give it a
  one-use research bonus.

DECISIONS
- Right-click a focus to add a decision unlocked by that focus.
- Use + Standalone Decision for a decision that does not require a focus.
- Set PP cost, countdown days, reward, manpower change, or temporary/permanent
  recruitable-manpower modifier.
- Add as many separate decisions to one focus as needed.

CHARACTERS
- Right-click a focus to add a recruited character, or use the Characters tab.
- Import a portrait and choose General, Admiral, or Officer.
- A character can be recruited by either a focus or a decision.

EVENTS
- Events can be created as country events or news events.
- Edit the trigger, picture, description, options and effects.
- Attached events are automatically triggered by their focus when exported.
- Upload a picture to preview normal and newspaper-style events.

ICONS
- Drop a PNG/JPG/WebP in the converter.
- Resize and position it inside the gold 95x85 crop box.
- Create Icon saves PNG and DDS copies. Use GFX_<icon name> as the focus icon key.
- Export copies converted DDS files and creates their GFX registration.

The bundled base_mod is an untouched copy of v0.76 used only as an export base.

Country and News event editors now use HOI4-matched visual frames. Adjust picture fit, scale, X and Y directly in the preview. Export uses 210x176 for country events and 397x153 for news events.


PRIVACY
- The public build contains no user project files, account names, home-directory paths, IP addresses, API keys, tokens, or analytics.
- Help → Check for updates reads public GitHub Releases, shows release notes, and verifies the Windows ZIP against its published SHA-256 checksum before installation. Download and installation require separate confirmation.
- Updates replace program files only. Projects, exports, backups, imports, selected paths, settings, and downloaded update files are preserved. A manual verified ZIP option is available when automatic download is unavailable.
- The program does not contact the developer's computer. Online updates, when enabled later, will only use public GitHub release endpoints.
- Projects and exports remain local on the user's computer.
